# 🎮 Tema Moderno RootGames - Gaming Marketplace

## 📋 **Visão Geral**

Criamos um tema moderno e atrativo para a plataforma RootGames, transformando a interface de um tema
básico para um design vibrante e adequado para uma marketplace de jogos.

## 🎨 **Características do Novo Tema**

### **Paleta de Cores Moderna**

- **Primária**: `#8b5cf6` (Roxo vibrante)
- **Secundária**: `#f231a5` (Rosa neon)
- **Acento**: `#0ea5e9` (Azul ciano)
- **Sucesso**: `#10b981` (Verde esmeralda)
- **Aviso**: `#f59e0b` (Laranja âmbar)
- **Erro**: `#ef4444` (Vermelho vibrante)

### **Gradientes Especiais**

- **Gradiente Principal**: `linear-gradient(135deg, #8b5cf6 0%, #f231a5 100%)`
- **Gradiente Escuro**: `linear-gradient(135deg, #1e1b4b 0%, #312e81 100%)`

- Cards com sombras
- Botões com animações
- Formulários modernos
